# Simple Outlook GUI

## Project Overview
Simple Outlook GUI is a user-friendly email client application built using Python's Tkinter library. It allows users to send and receive emails seamlessly, managing their communication in a straightforward and efficient manner.

## Features
- **Email Composition**: Write and send emails with rich text formatting.
- **Inbox Management**: View and manage incoming emails easily.
- **Contact Management**: Save and retrieve contacts for quick access.
- **Search Functionality**: Quickly search through your emails and contacts.
- **Multi-Account Support**: Manage multiple email accounts simultaneously.

## Installation
1. Ensure you have Python 3.x installed on your system.
2. Install the necessary packages using pip:
   ```bash
   pip install tkinter smtplib email
   ```
3. Clone the repository:
   ```bash
   git clone https://github.com/GirikJindal/Simple-Outlook-GUI.git
   ```
4. Navigate to the project directory:
   ```bash
   cd Simple-Outlook-GUI
   ```

## Usage
1. Run the application:
   ```bash
   python main.py
   ```
2. Follow the on-screen prompts to log in and start managing your email.

## Requirements
- Python 3.x
- Tkinter (comes pre-installed with Python)
- smtplib (standard for Python email handling)
